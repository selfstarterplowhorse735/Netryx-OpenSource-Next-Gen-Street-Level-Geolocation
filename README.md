## 🧭 What Netryx Does

Netryx is a local geolocation tool for street-level images. You give it an image, and it helps estimate the real-world location from visible clues like road shape, signs, buildings, terrain, and lighting.

It runs on your own Windows PC, so you keep full control of your files. It is built for people who want a desktop tool that works without a cloud account or web upload flow.

## ✨ Main Uses

- Find likely coordinates from a street photo
- Review image clues in one place
- Work offline after setup
- Keep files on your local machine
- Save time when checking locations from images
- Compare multiple image results in a simple view

## 🖼️ How to Use Netryx

### 1. Add an image
Open Netryx and load a street-level photo from your computer.

### 2. Let the tool scan it
Netryx checks the image for visual clues such as:
- Road markings
- Traffic signs
- Building styles
- Trees and terrain
- Shadow direction
- Camera view angle

### 3. Review the result
The app shows one or more likely locations, along with confidence signals and matched clues.

### 4. Compare and refine
If the first result looks close, use the visible clues to narrow it down. If not, try a different image or a better crop.

## 🔧 Best Results Tips

- Use images with signs, road labels, or landmarks
